# routine4ts

## 环境搭建

本 typescript 开发环境采用 rollup 生态库进行编译及打包，通过 execa 进行指令行扩展，用于后续对 rollup 指令的其它配置集成，tsconfig.json 文件中声明 typescript 编译选项，rollup.config.js 配置工程打包的选项。最终编译后的文件生成至工程根目录下的 dist 文件夹下。
